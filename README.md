# csfrn.portfolio

Personal portfolio website built with **Next.js (App Router)** + **TypeScript**, styled with **Tailwind CSS**, and set up for a modern component workflow (includes a `components.json`, commonly used with shadcn/ui). :contentReference[oaicite:0]{index=0}

## What’s in here

- `components/` — reusable UI components :contentReference[oaicite:2]{index=2}  
- `hooks/` — custom React hooks :contentReference[oaicite:3]{index=3}  
- `lib/` — helpers, utilities, shared logic :contentReference[oaicite:4]{index=4}  
- `public/` — static assets :contentReference[oaicite:5]{index=5}  
- `styles/` — global styles (if any) :contentReference[oaicite:6]{index=6}
